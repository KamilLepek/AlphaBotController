# AlphaBotController

AlphaBot User Manual: https://www.mouser.com/pdfdocs/AlphaBot-User-Manual2.pdf

Remote.py - Script used to control remotely AlphaBot platform connected to Raspberry Pi 3 Model B

Usage:  
  python Remote.py forward [duration]  
  python Remote.py backward [duration]   
  python Remote.py left [duration]  
  python Remote.py right [duration]  
  python Remote.py cleanup  
  python Remote.py stop  
  python Remote.py faster  
  python Remote.py slower  
  python Remote.py max_speed  
  python Remote.py help
